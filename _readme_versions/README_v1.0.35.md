# 班级管理系统 - 使用手册

中职班级综合管理 Web 系统，适用于班主任日常管理。所有数据本地存储，无需联网。

---

## 快速启动

### macOS

```bash
# 双击「启动_macOS.command」或在终端运行：
cd ~/Desktop/班级管理
bash 启动_macOS.command
```

### Windows

```bash
# 双击「启动_Windows.bat」即可，或在命令提示符运行：
cd C:\Users\你的用户名\Desktop\班级管理
启动_Windows.bat
```

首次运行会自动创建虚拟环境并安装依赖，后续启动更快。

启动后终端显示：

```
✅ 班级管理系统启动成功！
🌐 请访问: http://localhost:5800
🔑 管理员账号: admin / admin123
```

打开浏览器访问 **http://localhost:5800** 即可使用。

### 手动安装（可选）

```bash
cd ~/Desktop/班级管理
python3 -m venv venv
source venv/bin/activate        # macOS
# venv\Scripts\activate         # Windows
pip install flask flask-sqlalchemy openpyxl xlrd reportlab matplotlib pillow waitress
python3 app.py
```

- 首次使用：**admin** / **admin123**
- 注册新账号：登录页点击「注册账号」→ 用户名+密码即可

## 学期管理

折叠在侧边栏底部，点击「学期管理」展开：

| 操作 | 说明 |
|------|------|
| **切换学期** | 点击学期名称按钮 |
| **新建学期** | 弹窗：填学年起止+学期数（第1/2学期），自动生成名称 |
| **重命名** | 每个学期旁的编辑图标 |
| **删除** | 每个学期旁的×图标，仅有多学期时显示 |
| **数据隔离** | 各学期数据完全独立 |

### 新建学期格式
- 填起始年（如2026）→ 结束年自动+1（2027）
- 选第1学期或第2学期
- 自动生成名称：`2026-2027学年度第1学期`

## 学生管理

### 数据列（14列）
☑ 姓名 | 性别 | 身份证号 | 监护人 | 关系 | 电话 | 就读 | 建档 | 宿舍 | 户口 | 民族 | 住址 | 操作

- 所有字段独立列显示，不合并
- 全选后表格上方出现批量操作栏

### 批量操作
- **批量删除**：勾选学生 → 点「批量删除」→ 确认
- **流失标记**：勾选学生 → 点「流失」→ 弹窗填写每位学生的流失原因（必填）→ 确认

### 快捷编辑
- **就读方式**：下拉框切换（住校/走读），自动AJAX保存
- **建档立卡**：下拉框切换（否/贫困）
- **宿舍号**：输入框，失焦自动保存（走读时隐藏，显示「—」）

### 统计卡片（可点击筛选）
在籍人数 | 贫困户 | 住校生 | 通校生 | 少数民族 | 农业户口 | 非农业户口 | 特殊家庭 | 特异体质 | **流失**

- **流失**：标记为流失的学生不计入其他统计，点击「流失」卡片查看详情（含流失原因）
- 所有统计只计算在籍学生

---

## 每日考勤

- 每天 11 课时（早自习+第1~7节+夜自习1~3）
- 流失学生不显示、不参与考勤，历史数据保留
- 考勤周视图/考勤统计均过滤流失学生

---

## 违纪管理（折叠菜单）

### 处分管理
- 添加处分：选学生 → 日期/地点/事由/处分决定 → 选期限
- 多图上传，到期30天预警

### 违纪记录
- 添加违纪记录：选学生 → 日期/事由 → 反省时长下拉（1~15天）

---

## 班费管理

- 收入/支出录入，凭据图片上传
- 统计卡片：总收入 / 总支出 / 结余
- 导出：XLSX明细

---

## 页面优化

- **每页顶部居中**显示当前学期名称
- **学期切换** → 顶部学期名称自动更新
- **FontAwesome CDN** 加载图标，无本地字体路径问题
- **waitress 生产服务器** 替代 Flask 开发服务器（16线程并发）

## 技术栈

| 层级 | 技术 |
