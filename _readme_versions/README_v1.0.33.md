# 班级管理系统 - 使用手册

中职班级综合管理 Web 系统，适用于班主任日常管理。所有数据本地存储，无需联网，多用户独立数据库，加密备份。

## 快速启动

### macOS / Linux

```bash
cd 班级管理
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python3 app.py
# 访问 http://localhost:5800
```

首次运行自动创建管理员：**admin / admin123** 和默认学期。

## 多用户系统

支持多教师独立使用，每用户拥有完全独立的数据库。

```
instance/
├── master.db           # 登录信息（用户名+密码）
└── users/
    ├── u1.db           # admin 的业务数据
    ├── u2.db           # teacher1 的业务数据
    └── ...
```

- 注册新用户后自动创建独立的业务数据库
- 用户之间数据完全隔离，互不可见
- 登录时自动切换到对应用户的数据库

## 数据管理

提供全量数据导出、加密备份和版本化恢复功能。

| 功能 | 说明 |
|------|------|
| 全量导出 | 直接下载当前用户的 `.db` 数据库文件 |
| 加密备份 | 生成 `.cmb` 格式加密备份（Fernet 加密，用密码哈希派生密钥） |
| 备份列表 | 查看所有历史备份，支持下载和删除 |
| 数据恢复 | 上传 `.cmb` 或 `.db` 文件恢复数据，自动验证用户身份 |
| 前向兼容 | 旧版本备份恢复时自动运行 schema 迁移，补全缺失列 |

| 模块 | 页面 | 功能 |
|------|------|------|
| **仪表盘** | `/` | 统计卡片（在籍/贫困户/住校/通校/少数民族等）、到期预警 |

| **座位管理** | `/seat` | 初始化座位网格、拖拽换座、自动排座 |
| **处分管理** | `/discipline` | 多图上传、到期预警（30天）、导出 |
| **违纪记录** | `/violation` | 1-15天反省、到期提醒 |
| **班费管理** | `/fund` | 收支记录、凭证、XLSX导出 |
| **学期管理** | 侧边栏 | 模板新建、重命名、删除（数据严格隔离） |

## 数据安全

- **本地存储**: 所有数据存本地 SQLite，不依赖外部服务
- **多用户隔离**: 每用户独立数据库文件
- **加密备份**: `.cmb` 格式 + Fernet 加密，仅当前用户可解密
- **前向兼容**: 备份文件记录 schema 版本，恢复时自动迁移
- **学期隔离**: 不同学期数据完全分离

---

## 技术栈

- **后端**: Python 3 + Flask + Flask-SQLAlchemy + SQLite
- **前端**: Bootstrap 5 + FontAwesome + jQuery
- **导出**: openpyxl（XLSX）、cryptography.fernet（加密备份）
- **部署**: waitress（生产服务器）
- **NAS 部署**: fnOS fpk 格式（fnpack 构建）
