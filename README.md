# 班级管理系统

中职班级综合管理 Web 系统，用于班主任日常管理工作。涵盖学生信息管理、每日考勤、成绩分析、实训记录、班级课表、座位安排、处分管理、违纪记录、班费收支、学期管理等核心功能。支持多用户独立数据库，所有数据本地存储，无需联网。

**版本**: v1.0.37  
**作者**: 万钟  
**源码**: https://github.com/Jwzhonga/class-manager  
**下载**: https://github.com/Jwzhonga/class-manager/releases/tag/v1.0.37

# 快速开始

## 本地运行

```bash
# 克隆仓库
git clone https://github.com/Jwzhonga/class-manager.git
cd class-manager

# 安装依赖
pip install -r requirements.txt

# 启动（首次运行自动创建管理员）
python3 app.py

# 打开浏览器访问 http://localhost:5800
# 默认管理员: admin / admin123
```

推荐使用 waitress 生产服务器：
```bash
pip install waitress
python3 -c "from waitress import serve; from app import app; serve(app, host='0.0.0.0', port=5800, threads=16)"
```

## 飞牛 NAS 部署

从 [Releases](https://github.com/Jwzhonga/class-manager/releases) 下载 `Classmanager.fpk`，在飞牛应用中心手动安装。安装过程自动下载 Python 依赖（清华镜像源），数据库持久化在应用数据目录，升级不丢失数据。

# 功能详解

## 1. 多用户系统

支持多教师独立使用，每用户拥有完全独立的数据库。

```
instance/
├── master.db           # 登录信息（用户名+密码哈希）
└── users/
    ├── u1.db           # 用户1的业务数据（含全部17张表）
    ├── u2.db           # 用户2的业务数据（完全隔离）
    └── ...
```

- 注册后自动创建独立业务数据库和默认学期
- 用户之间数据完全隔离，互不可见
- 登录时自动切换到对应用户的数据库引擎

---

## 2. 学生管理

**路径**: `/students`

学生信息管理核心页面，支持完整的增删改查操作。

### 表格显示
- 14 列独立显示，不合并：姓名、性别、身份证号、监护人、关系、电话、就读方式、建档立卡、宿舍号、户口性质、民族、家庭住址、操作、多选复选框
- 所有表头支持点击排序（按文字颜色深浅指示排序列）
- 列宽自适应，全部 12px 字体

### 批量操作
- 勾选多个学生后，顶部显示批量操作栏
- **批量删除**: 勾选后点击删除，一次性删除多个学生
- **批量流失**: 勾选后点击流失，弹窗要求填写每位学生的流失原因（必填）
- 全选/取消全选复选框

### 快捷编辑
- 就读方式下拉框：住校/走读，切换时自动联动宿舍号显示/隐藏
- 建档立卡下拉框：贫困/低保/特困/残疾/孤儿/烈士子女/其他
- 宿舍号输入框（仅住校显示）
- 所有快捷编辑通过 AJAX 异步保存，无需刷新页面

### 筛选与统计
- 首页统计卡片：在籍人数、贫困户数、住校生数、通校生数、少数民族数、城镇户口数、农村户口数、特殊家庭类型数、特异体质人数
- 点击统计卡片可筛选对应学生
- 流失生（status='withdrawn'）不参与任何统计，仅在流失卡片中展示详情（含流失原因）
- 支持筛选：全部学生、流失学生、特殊家庭、特异体质

### 数据导入
- 支持 Excel 导入学生花名册
- 去重依据：姓名 + 身份证号 + 学期

### 数据导出
- 特殊家庭名单 XLSX
- 宿舍名单 XLSX
- 违纪记录 XLSX

## 3. 每日考勤

**路径**: `/attendance`

### 考勤策略
- 每人每天一条记录（非每节课一条）
- 11 课时：早自习 + 第1~7节 + 夜自习1/2/3
- 课时用 bitmask 存储（2^0 ~ 2^10 = 1~1024，全选 = 2047）
- 默认全天出勤 = 11 节

### 考勤状态
| 状态 | 标记颜色 | 说明 |
|------|---------|------|
| 出勤 | 🟢 绿色 | 正常到课 |
| 迟到 | 🟡 黄色 | 迟到 |
| 病假 | 🔵 蓝色 | 因病请假 |
| 事假 | 🟡 黄色 | 因事请假 |
| 旷课 | 🔴 红色 | 无故缺课 |

### 课时选择
- 下拉菜单多选：全选/取消、早自习、第1~7节、夜自习1/2/3
- 下拉框显示已选节数（如"5节"）

### 统计与视图
- **每日考勤**: 当天考勤录入界面，支持图片凭证上传
- **考勤周视图** (`/attendance/weekly`): 按周展示学生出勤概览，支持上一周/本周/下一周导航
- **考勤统计** (`/attendance/stats`): 出勤率统计、病假事假统计、图表展示

## 4. 成绩管理

**路径**: `/grades`

### 评分体系
- 每科目 4 项评分：课堂表现、作业质量、课堂笔记、考试成绩
- 权重设置：0.2 / 0.2 / 0.2 / 0.4
- 支持按科目筛选、添加考试记录
- 成绩分析功能：统计图表

---

## 5. 任课管理

**路径**: `/teaching`

跨班级科目和学生的独立管理模块。

- 添加任课科目：科目名称、任课教师、班级名称
- 科目卡片展示：科目名、教师名、班级名、学生人数
- 学生管理：
  - 文本框批量导入（每行一个姓名）
  - Excel 导入（第一列为学生姓名）
  - 单个学生删除（退学等情况）
- 学生数据与学生管理模块中的 Student 表无关，独立管理
- 支持删除整门科目及关联学生数据

---

## 6. 实训管理

**路径**: `/training`

### 实训项目
- 自定义实训项目名称
- 按学生逐个记录实训完成情况
- 完成质量评级

### 实训分组 (`/training/groups`)
- 创建分组，为分组添加成员
- 添加实训项目时可按分组添加，自动为该组全部成员创建记录
- 已分配到其他分组的学生在添加页面自动过滤（每个学生只能在一个分组中）
- 删除分组不影响历史实训记录

## 7. 班级课表

**路径**: `/schedule`

- 支持多个 Sheet（课表页签）
- 每个 Sheet 独立管理课程安排
- 柔和浅色系填充，每门课不同颜色
- 课程单元格 3D 阴影凸起效果
- 圆角单元格
- 点击课程高亮同名课程
- 支持 xlsx 导入自动解析

## 8. 座位管理

**路径**: `/seat`

- 初始化座位网格（输入行数 x 列数）
- HTML5 Drag and Drop 拖拽换座
- 已安排座位显示学生姓名
- 空座位可点击选择学生
- 自动排座：随机安排所有未分配学生
- 清空所有座位安排
- 学期隔离：每学期独立座位布局

## 9. 处分管理

**路径**: `/discipline`

- 处分类型：警告、严重警告、记过、记大过、留校察看、开除学籍
- 处分期限：1个月 / 3个月 / 半年 / 1年 / 自定义
- 到期前 30 天在首页仪表盘预警提示
- 支持多张通知照片上传（JSON 数组存储）
- 导出带嵌入图片

## 10. 违纪记录

**路径**: `/violation`

- 违纪类型：抽烟、喝酒、打架、玩手机、迟到、早退、旷课、上课睡觉、未戴校牌、仪容仪表、顶撞老师、其他
- 反省天数：1~15 天
- 到期自动提醒

## 11. 班费管理

**路径**: `/fund`

- 收入/支出记录
- 收支凭证图片上传
- 收支汇总统计
- XLSX 导出（带金额格式、求和公式）

## 12. 数据管理

**路径**: `/data`

### 全量导出
- 直接下载当前用户的完整数据库文件（.db）
- 注意：下载后请勿在应用运行时覆盖数据库

### 加密备份
- 生成 `.cmb` 格式加密备份文件（ClassManager Backup）
- 备份格式：`CMB1|用户ID|时间戳|schema版本号\n<加密数据>`
- 加密方式：Fernet 对称加密，密钥从用户密码哈希派生
- 备份文件存储于 `static/backups/` 目录

### 备份列表
- 展示所有历史备份：文件名、文件大小、创建时间
- 支持下载备份文件
- 支持删除备份文件
- 仅显示当前用户的备份

### 数据恢复
- 支持上传 `.cmb`（加密格式）或 `.db`（明文格式）文件恢复
- 恢复前自动验证：备份文件中的用户 ID 必须匹配当前登录用户
- 恢复前自动备份当前数据库（防止误操作）
- 前向兼容：旧版本备份恢复时自动检测并补全缺失的列

---

## 13. 学期管理

侧边栏折叠菜单。

### 新建学期
- 模板格式：选择起始年 + 学期数（第1学期/第2学期）
- 名称自动生成：`XXXX-XXXX学年度第X学期`
- 结束年 = 起始年 + 1
- 起止日期可单独设置

### 学期操作
- 折叠菜单展示所有学期
- 单击切换学期
- 重命名（弹窗修改）
- 删除（仅剩一个学期时隐藏删除按钮）

### 数据隔离
- 不同学期的数据完全独立
- 删除学期时清理全部关联数据（17 张表的关联记录）
- 当前学期的数据影响首页统计和各模块展示

---

## 14. 页面通用设计

- **固定侧边栏**: 浅色主题（默认极光蓝），220px 宽，独立滚动，底部含主题颜色选择器（6 种预设）
- **粘性页面标题**: 顶部居中显示当前学期名称
- **统计卡片**: 直观展示关键数据，2 列排列，点击可筛选
- **响应式布局**: 全面优化手机端体验（紧凑卡片、表格卡片模式、2 列布局、主题切换支持）
- **本地静态资源**: Bootstrap 5 + FontAwesome 6 CSS/JS/字体内嵌在 `static/` 目录，不依赖 CDN
- **当前学期显示**: 每页顶部居中显示当前学期名称

# 数据安全

| 机制 | 说明 |
|------|------|
| 本地存储 | 所有数据存本地 SQLite，不依赖外部服务或云平台 |
| 多用户隔离 | 每用户独立数据库文件，互不可见 |
| 加密备份 | `.cmb` 格式 + Fernet 加密，仅当前用户可解密恢复 |
| 前向兼容 | 备份文件记录 schema 版本，恢复时自动运行 ALTER TABLE 迁移 |
| 学期隔离 | 不同学期数据完全分离，删除学期时级联清理 17 张关联表 |
