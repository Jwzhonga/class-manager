# 班级管理系统 - 使用手册

中职班级综合管理 Web 系统，适用于班主任日常管理。所有数据本地存储，无需联网。

---

## 快速启动

### macOS

```bash
# 双击「启动_macOS.command」或在终端运行：
cd ~/Desktop/班级管理
bash 启动_macOS.command
```

### Windows

```bash
# 双击「启动_Windows.bat」即可，或在命令提示符运行：
cd C:\Users\你的用户名\Desktop\班级管理
启动_Windows.bat
```

首次运行会自动创建虚拟环境并安装依赖，后续启动更快。

启动后终端显示：

```
✅ 班级管理系统启动成功！
🌐 请访问: http://localhost:5800
🔑 管理员账号: admin / admin123
```

打开浏览器访问 **http://localhost:5800** 即可使用。

### 手动安装（可选）

```bash
cd ~/Desktop/班级管理
python3 -m venv venv
source venv/bin/activate        # macOS
# venv\Scripts\activate         # Windows
pip install flask flask-sqlalchemy openpyxl xlrd reportlab matplotlib pillow
python3 app.py
```

## 学生管理

### 导入学生（Excel）
1. 点击「导入Excel」
2. 选择 .xls 或 .xlsx 文件（前2行为标题，第3行起为数据，第2列为姓名）
3. 导入后自动分配到当前学期

### 手动添加
点击「+ 添加学生」→ 填写信息 → 保存

### 统计卡片（可点击筛选）
页面顶部：学生总数、贫困户、住校/通校、少数民族、户口类型、年龄分布、户籍地。

**新增统计项：**
- 🔴 **特殊家庭** — 点击显示类型分布（孤儿/留守/单亲等）+ 对应学生姓名
- 🟢 **特异体质** — 点击筛选特异体质学生，下方显示姓名+具体情况

### 新增学生字段
- **宿舍号** — 选「住校」后显示输入框
- **特殊家庭** — 7类勾选框（单亲/留守/离异/孤儿/残疾/贫困/其他）+ 说明框
- **特异体质** — 是/否下拉 + 说明框
- **备注**

### 导出
- **导出特殊家庭台账** — 匹配教育局模板格式
- **导出宿舍名单** — 按宿舍号排序

### 表格
- 列宽：拖拽表头边框调整
- 排序：点击表头排序（箭头已隐藏，通过文字颜色深浅标识排序列）

## 每日考勤

### 课时安排
每天 **11 个课时**：早自习 + 第1~7节 + 夜自习1~3

### 操作
1. **选择日期**（右上角日期选择器，自动切换）
2. **一键全勤**：绿色按钮，全部出勤+全天，自动保存
3. **单独修改**：学生状态下拉框（出勤/迟到/病假/事假/旷课）
4. **选择缺勤课时**：点击「缺勤课时」按钮 → 弹出复选框勾选（可多选，有全选）
5. **填写原因**：非出勤状态可填原因
6. **上传凭证**：支持 jpg/png/gif
7. **保存** → 自动跳转回当天

### 课时选择逻辑
- 出勤 → 自动全选=全天（11节），按钮显示「全天」
- 迟到 → 默认选早自习
- 请假/旷课 → 手动勾选

### 右侧登记栏
实时显示当天已登记的考勤记录。

## 违纪管理（侧边栏折叠菜单）

包含两个子项：

### 处分管理
- 添加处分：选学生 → 日期/地点/事由/处分决定 → 选期限（1月/3月/半年/1年/自定义）
- 上传凭证照片（可多张，JSON数组存储）
- **到期预警**：首页蓝条提示30天内到期的处分
- 编辑/删除：每行右侧按钮
- 导出：Excel/PDF（含嵌入照片）

### 违纪记录（新增）
- 添加违纪记录：选学生 → 日期/事由 → **反省时长下拉框（1~15天）** → 自动计算起止日期
- 顶部 badge 显示各反省天数的记录数分布
- 导出违纪明细（按日期倒序）
- 导出违纪统计（按反省天数汇总+占比）

## 性能优化

- **gzip 压缩** — 响应超过1KB自动压缩
- **静态文件缓存** — 1年缓存期（SEND_FILE_MAX_AGE_DEFAULT）
- **waitress 多线程** — 生产模式下16线程并发
- **CDN 资源** — FontAwesome/Bootstrap 从 bootcdn 加载
- **数据库** — SQLite，学期隔离，fnOS 上存 TRIM_PKGVAR 保证升级不丢数据

## 数据备份

数据库文件在 `instance/` 目录下（fnOS 上存于持久化数据目录）。

```bash
# macOS
cp instance/class_manager.db ~/Desktop/班级备份_$(date +%Y%m%d).db

# Windows
copy instance\class_manager.db 班级备份_%date:~0,10%.db
```

恢复时复制回 `instance/class_manager.db`（或对应路径）重启即可。

## 常见问题

**Q: 无法访问 http://localhost:5800？**
A: 确认终端在运行。端口被占用时运行 `python3 app.py --port 5801`。

**Q: 图标不显示？**
A: 确保网络能访问 bootcdn.net。如果内网受限，将 base.html 中的 CDN 链接改为本地静态文件。

**Q: 导入 Excel 显示 0 条？**
A: 表头从第3行开始，第2列为姓名。

**Q: 忘记密码？**
A: 直接注册新账号，或运行：
```bash
python3 -c "from app import app, db, User; u=User.query.filter_by(username='admin').first(); u.set_password('新密码'); db.session.commit()"
```

**Q: 考勤页面报 500 错误？**
A: 确认 `app.jinja_env.filters['popcount']` 已注册（检查 app.py 第 70 行左右）。
