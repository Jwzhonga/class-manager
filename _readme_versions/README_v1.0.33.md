# 班级管理系统 - 使用手册

中职班级综合管理 Web 系统，适用于班主任日常管理。所有数据本地存储，无需联网。

---

## 快速启动

### macOS

```bash
# 双击「启动_macOS.command」或在终端运行：
cd ~/Desktop/班级管理
bash 启动_macOS.command
```

### Windows

```bash
# 双击「启动_Windows.bat」即可，或在命令提示符运行：
cd C:\Users\你的用户名\Desktop\班级管理
启动_Windows.bat
```

首次运行会自动创建虚拟环境并安装依赖，后续启动更快。

启动后终端显示：

```
✅ 班级管理系统启动成功！
🌐 请访问: http://localhost:5800
🔑 管理员账号: admin / admin123
```

打开浏览器访问 **http://localhost:5800** 即可使用。

### 手动安装（可选）

```bash
cd ~/Desktop/班级管理
python3 -m venv venv
source venv/bin/activate        # macOS
# venv\Scripts\activate         # Windows
pip install flask flask-sqlalchemy openpyxl xlrd reportlab matplotlib pillow
python3 app.py
```

## 学期管理

每个学期拥有完全独立的数据，切换学期互不干扰。

| 操作 | 位置 | 说明 |
|-----|------|------|
| **切换学期** | 左侧底部下拉框 | 点击即可切换 |
| **新建学期** | 侧边栏「+ 新建学期」 | 填写名称+起止日期，自动切换过去 |
| **删除学期** | 下拉框右侧红色 × | 两次确认后删除学期+所有关联数据 |

## 学生管理

### 导入学生（Excel）
1. 点击「导入Excel」
2. 选择 .xls 或 .xlsx 文件（前2行为标题，第3行起为数据，第2列为姓名）
3. 导入后自动分配到当前学期

### 手动添加
点击「+ 添加学生」→ 填写信息 → 保存

### 统计卡片
页面顶部：学生总数、贫困户、住校/通校、少数民族、户口类型、年龄分布、户籍地。**点击卡片可筛选**。

### 表格
- 列宽：拖拽表头边框调整
- 排序：点击表头切换

## 每日考勤

### 课时安排
每天 **11 个课时**：早自习 + 第1~7节 + 夜自习1~3

### 操作
1. **选择日期**（右上角）
2. **一键全勤**：绿色按钮，全部出勤+全天，自动保存
3. **单独修改**：学生状态下拉框（出勤/迟到/病假/事假/旷课）
4. **选择缺勤课时**：点击「缺勤课时」按钮 → 弹出复选框勾选（可多选，有全选）
5. **填写原因**：非出勤状态可填原因
6. **上传凭证**：支持 jpg/png/gif
7. **保存**

### 课时选择逻辑
- 出勤 → 自动全选=全天（11节），按钮显示「全天」
- 迟到 → 默认选早自习
- 请假/旷课 → 手动勾选

### 右侧登记栏
实时显示当天已登记的考勤记录。

## 考勤周视图

左侧菜单「考勤周视图」→ 周历表格展示本周每天每人的考勤状态：
- ✅ 出勤（绿）、旷X节（红）、病X节（黄）、事X节（蓝）、迟X节（橙）

## 考勤统计

左侧菜单「考勤统计」→ 缺勤趋势柱状图 + 每人病/事/旷/迟到统计。

## 成绩管理

### 科目
页面底部添加/删除科目（名称+教师姓名）

### 录入
1. 顶部选科目
2. 每名学生四维评分：课堂表现20% + 作业质量20% + 课堂笔记20% + 考试成绩40%
3. 保存 → 自动算综合分

### 分析
左侧「成绩分析」→ 各科平均分

---

## 实训管理

### 项目
添加/删除实训项目（名称、描述、满分、类别）

### 记录
选学生+项目 → 填完成质量、分数、日期、评语

### 统计
「实训统计」→ 每位学生各项目完成情况矩阵

---

## 处分管理

### 添加
选学生 → 填日期/地点/事由/处分决定 → 选期限（1月/3月/半年/1年/自定义） → 上传凭证照片（可多张） → 备注

### 到期预警
首页蓝条提示30天内到期的处分

### 编辑/删除
每行右侧按钮

---

## 班级课表

### 手动添加
选星期+节次+课程名+教师+教室

### Excel上传
支持多sheet选择，自动解析上午/下午节次

### 拖拽调整
课程可拖拽到其他位置，自动保存

### 样式
每门课独立柔和配色，点击高亮同名课程（3D阴影）

## 数据导出

所有导出按钮在各页面**右上角**，支持筛选（姓名+日期范围）。

| 页面 | 导出类型 |
|-----|---------|
| 每日考勤 | 汇总Excel、明细Excel/PDF |
| 考勤统计 | 汇总/明细Excel |
| 处分管理 | Excel/PDF（含嵌入照片） |
| 成绩管理 | Excel/PDF（四维评分表） |
| 实训管理 | Excel/PDF |
| 班级课表 | Excel/PDF |

- **PDF**：每页最多2张图片，400×300
- **Excel**：图片按实际比例缩放，最大300px

## 常见问题

**Q: 无法访问 http://localhost:5800？**
A: 确认终端在运行。端口被占用时修改 `app.py` 末行的 `port=5800`。

**Q: 导入 Excel 显示 0 条？**
A: 表头从第3行开始，第2列为姓名。

**Q: 忘记密码？**
A: 直接注册新账号，或运行：
```bash
python3 -c "from app import app, db, User; u=User.query.filter_by(username='admin').first(); u.set_password('新密码'); db.session.commit()"
```
